# NPN Common-Emitter Amplifier Simulation

This project presents the design and simulation of an NPN common-emitter amplifier using PSpice.

## Topics Covered
- NPN transistor amplifier design
- Simulated output waveform
- DC operating values
- Output resistance
- Voltage gain

## Software
- PSpice

## Summary
This project demonstrates the operation of a common-emitter amplifier. The simulation includes the circuit schematic, output waveform, calculated output resistance, and voltage gain analysis.
