# CMOS Inverter Design and Simulation

This project demonstrates the design and simulation of a CMOS inverter using PMOS and NMOS transistors in Logisim.

## Topics Covered
- CMOS inverter circuit
- PMOS and NMOS switching
- Logic inversion
- Truth table verification

## Software
- Logisim

## Summary
This project verifies the behavior of a CMOS inverter for both input states. It shows how complementary MOS transistors work together to produce correct logic inversion.
