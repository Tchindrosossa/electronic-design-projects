# Electronic Design Projects

This repository contains my analog and digital electronics design projects completed in ELEC 359. These projects include transistor I-V characteristic analysis, common-emitter amplifier simulation, and CMOS inverter design using PSpice and Logisim.

## Projects

### 1. BJT I-V Characteristics Simulation
Analysis of NPN and PNP transistor current-voltage characteristics using PSpice.

### 2. NPN Common-Emitter Amplifier Simulation
Design and simulation of a common-emitter amplifier including DC values, output resistance, and voltage gain.

### 3. CMOS Inverter Design and Simulation
Design and simulation of a CMOS inverter using PMOS and NMOS transistors in Logisim.

## Tools Used
- PSpice
- Logisim
- Analog and Digital Electronics concepts
