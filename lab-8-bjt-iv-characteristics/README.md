# BJT I-V Characteristics Simulation

This project analyzes the current-voltage characteristics of NPN and PNP transistors using PSpice.

## Topics Covered
- NPN transistor characteristics
- PNP transistor characteristics
- IC vs VBE
- IC vs VCE
- IC vs VEB
- IC vs VEC

## Software
- PSpice

## Summary
This simulation study shows how collector current changes with voltage in both NPN and PNP transistors. It helps demonstrate the electrical behavior of bipolar junction transistors under different biasing conditions.
